A Note on Deprecated / Obsolete Elements

Many older HTML tags have been deprecated in HTML5 because their function was purely presentational. Styling should now be handled by CSS.

Avoid using tags such as:

  • <font>, <center>, <big>
  • <frame>, <frameset>, <noframes>
  • <applet>, <acronym>, <tt>, <strike>
  • <menuitem> (limited support), <keygen> (deprecated)

Use modern, semantic HTML5 elements and CSS for styling instead.
